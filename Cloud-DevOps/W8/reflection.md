# W8 Day 1 Reflection

## Terraform là gì?
Terraform là công cụ Infrastructure as Code (IaC) giúp quản lý hạ tầng bằng code.

## Hôm nay em đã học
- Terraform Init
- Terraform Validate
- Terraform Plan
- Terraform Apply
- Terraform State

## Điều em hiểu được
Terraform giúp tự động hóa việc triển khai hạ tầng và dễ dàng quản lý bằng Git.

## Điều em cần tìm hiểu thêm
- Terraform Modules
- Remote State
- Terraform Best Practices
